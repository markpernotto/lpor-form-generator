4. Available URL Parameters

Pre-populate court information fields via URL query parameters

Parameter Description Example
court Court name Caddo%20Parish%20Court
parishCity Parish or city name Caddo%20Parish
division Court division A, B, C
number Docket/case number 2024-CV-12345
filed Filing date (YYYY-MM-DD) 2024-10-16
clerk Clerk name John%20Doe
widget Enable widget mode (optional) true
debug Show test data button (dev only) true

Example URLs:

Basic: /en/lpor_f?court=Test%20Court&number=123
Complete: /en/lpor_f?court=Jefferson%20Parish&parishCity=Jefferson&number=2024-CV-001&division=B&filed=2024-10-16&clerk=Jane%20Smith
Spanish: /es/lpor_f?court=Tribunal%20Test&number=ESP-555&division=C
Debug Mode: /en/lpor_f?debug=true&court=Dev%20Court

6. Implementation Guide

How to embed this form on your website

Step 1: Copy the iframe code

<iframe src="https://your-domain.com/en/lpor_f?widget=true" width="100%" height="1000" style="border: none;" title="LPOR-F Form" loading="lazy"> </iframe>

Step 2: Customize with URL parameters

Add court information parameters to pre-populate fields:

https://your-domain.com/en/lpor_f?widget=true&court=My%20Court&number=2024-123&division=A&filed=2024-10-16

Step 3: Adjust height as needed

The form needs approximately:

  • 1000px for desktop full view
  • 1200px+ if form has many sections filled
  • 800px for compact/mobile views

Step 4: Consider responsive sizing

<style> .lpor-form-container { width: 100%; max-width: 1200px; margin: 0 auto; } .lpor-form-container iframe { width: 100%; height: 1000px; border: none; } @media (max-width: 768px) { .lpor-form-container iframe { height: 1200px; } } </style> <div class="lpor-form-container"> <iframe src="https://your-domain.com/en/lpor_f?widget=true" title="LPOR-F Form"></iframe> </div>
